Leia antes de jogar

A versão web do DarkMice é experimental e portanto pode conter problemas de desempenho ou bugs visuais nos menus do jogo. Não nos reporte estes problemas dado que eles vem do emulador flash Ruffle ao qual estamos usando para tornar isso possível. É uma questão de tempo até que os colaboradores do Ruffle aprimorem seu desempenho e suportem mais recursos, corrigindo os bugs encontrados. Para um melhor desempenho, recomenda-se jogar no modo minimalista.
